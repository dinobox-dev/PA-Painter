# 스트로크 밀도 제어 분석

**작성일**: 2026-02-22

---

## 현재 밀도 결정 파이프라인

스트로크 밀도는 4단계 파이프라인을 거치며 결정된다. (`path_placement.rs`)

### 1단계: 시드 그리드 생성 (`generate_seeds`)

```
spacing = brush_width / resolution × stroke_spacing
```

- 균일 그리드 위에 `spacing × 0.2` 반경의 원형 지터 적용
- `stroke_spacing`이 밀도의 **1차 제어 파라미터** (기본값 1.0)
- spacing을 반으로 줄이면 시드 수 4배

### 2단계: 경로 추적 (`trace_streamline`)

각 시드에서 방향장을 따라 경로를 추적하며, 아래 조건에서 조기 종료:

| 종료 조건 | 파라미터 | 기본값 |
|---|---|---|
| 목표 길이 도달 | `max_stroke_length` | 240px |
| 누적 꺾임 | `max_turn_angle` | 15° |
| UV 경계 도달 | — | [0,1]² |
| 색상 경계 | `color_break_threshold` | None |
| 노말 경계 | `normal_break_threshold` | None |

목표 길이는 `max_stroke_length × U^0.5` (power distribution)로 결정되어 중간 길이 스트로크가 다수를 차지한다.

### 3단계: 최소 길이 필터

```
min_length = brush_width / resolution
```

이보다 짧은 경로는 폐기. 브러시 폭 1배 미만의 "점" 형태 스트로크를 제거한다.

### 4단계: 중복 경로 필터 (`filter_overlapping_paths`)

```
threshold_dist = brush_width_uv × 0.3
overlap_ratio  = 0.7
```

이미 수락된 경로와 비교하여 포인트의 70% 이상이 `threshold_dist` 이내에 있으면 폐기.
공간 해시 그리드로 O(n·m·k) 탐색.

## 밀도 영향도 순위

| 순위 | 파라미터 | 영향 경로 | 비고 |
|---|---|---|---|
| 1 | `stroke_spacing` | 시드 그리드 간격 직접 제어 | 가장 직접적 |
| 2 | `brush_width` | 그리드 간격 + 중복 필터 거리 + 최소 길이 | 3중 효과 |
| 3 | `max_stroke_length` | 평균 스트로크 길이 → 커버리지 | 간접적 |
| 4 | `max_turn_angle` | 조기 종료 빈도 | 방향장 의존 |
| 5 | `color/normal_break_threshold` | 경계에서 경로 단절 | 선택적 |

## 밀도를 올리기 위한 개선 방향

### A. `stroke_spacing` 하향 (가장 단순)

- `stroke_spacing`을 줄이면 시드가 촘촘해진다.
- 단, 현재 중복 필터의 하드코딩 값이 병목이 되어 spacing을 아무리 줄여도 상한이 존재한다.
- **필요 조건**: 중복 필터 완화와 병행해야 효과적.

### B. 중복 필터 파라미터화

현재 하드코딩된 두 값을 `StrokeParams`로 노출:

- **`overlap_ratio`** (현재 0.7 고정): 올리면 더 많은 겹침 허용 (예: 0.85)
- **`threshold_dist` 계수** (현재 0.3 고정): 줄이면 "가까운 경로" 판정 범위가 좁아짐

이렇게 하면 `stroke_spacing` 하향과 조합하여 밀도 상한을 실질적으로 해제할 수 있다.

### C. 시드 생성 전략 개선

- **포아송 디스크 샘플링**: 균일 그리드보다 자연스러운 분포, 최소 간격 보장하면서 더 촘촘한 배치 가능.
- **빈 영역 감지 2패스**: 1패스 후 커버리지가 낮은 영역에 추가 시드 생성.
- 구현 복잡도가 올라가므로 B 방향과의 비용/효과 비교 필요.

### D. 멀티패스 레이어링

- 동일 레이어를 서로 다른 시드로 복수 패스 생성 후 합산.
- 중복 필터를 패스 내에서만 적용하면 패스 간 겹침이 자연스럽게 허용됨.
- 실제 유화의 여러 번 덧칠하는 과정과 유사한 모델.

## 추천 구현 순서

1. **B (중복 필터 파라미터화)** — 가장 적은 변경으로 밀도 상한 해제
2. **A (stroke_spacing 조정)** — B와 조합하여 즉시 효과 확인 가능
3. **D (멀티패스)** — 자연스러운 밀도 증가, 기존 구조 재활용 가능
4. **C (시드 전략)** — 장기적 품질 개선, 우선순위 낮음
